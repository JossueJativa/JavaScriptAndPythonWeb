Comandos a usar en Git

Comando para identificarse "email y nombre"

git config --global user.email "Email"

git config --global user.name "Nombre"

Comando para clonar repositorio

git clone 'url'

Comando para crear archivos en terminal git

touch

Comando para agregar archivos en git

git add 'FileName'

Agregar comentarios al repositorio de git

git commit -m "comentario"

Comando para ver los que se a hecho en el repositorio local

git status

Comando para enviar nuestro proyecto a GitHub

git push

Comando para guardar u enviar comentarios a la vez en git

git commit -am "Comentario"

Comando para descargar la ultima version de GitHub

git pull
